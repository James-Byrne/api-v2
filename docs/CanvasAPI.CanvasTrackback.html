    Canvas API v0.0.1 CanvasAPI.CanvasTrackback

    A “canvas trackback” is an event where a reference to a canvas was either added to or removed from another canvas.

    Summary

    Functions

    add(target_id, source_id, acct_id)

    Create an “added” reference event, where the account user added a reference to the target canvas to the source canvas

    remove(target_id, source_id, acct_id)

    Create a “removed” reference event, where the account user removed a reference to the target canvas from the source canvas
