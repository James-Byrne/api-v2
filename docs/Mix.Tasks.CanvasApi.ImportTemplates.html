    Canvas API v0.0.1 Mix.Tasks.CanvasApi.ImportTemplates

    Import templates from URLs. The URL must return a 200 status code and a body that is just an array of blocks represented as JSON.

    Examples 

    mix canvas_api.import_templates https://example.com/blocks.json

mix canvas_api.import_templates \
  https://example.com/blocks-1.json \
  https://example.com/blocks-2.json \
  https://example.com/blocks-3.json
